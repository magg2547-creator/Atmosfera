# Atmosfera

## Project Structure

- `api/` - Vercel serverless API routes.
- `src/` - Frontend HTML, CSS, and browser modules.
- `src/utils/` - Shared helper functions.
- `src/services/` - Data fetching, export, and dashboard orchestration.
- `src/charts/` - Chart.js plugins.
- `assets/` - Images and icons.
